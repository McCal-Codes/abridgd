# Archived / Deprecated Docs
Last Updated: 2026-01-26

Use this folder for retired, superseded, or experimental documents so the main docs tree stays clean. Keep a short note in each archived file about why it moved and the date.

## When to archive
- A doc is superseded by a newer guide or ADR.
- Experimental notes are no longer active but worth preserving.
- Release-specific docs that are no longer current (unless still linked for users).

## How to archive safely
1. Move the file into this folder (maintain relative subfolders if helpful).
2. Add a top-level banner: `> Archived on YYYY-MM-DD — superseded by ...` and link to the replacement.
3. Update indexes/links (README, per-folder indexes) to point to the new canonical doc.
4. If external links might break, leave a stub in the old location pointing here.
