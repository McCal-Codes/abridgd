# TestFlight Release Notes — Abridged v1.0.0
Last Updated: 2026-01-26

Thank you for joining the Abridged beta! This release includes our new iOS 26-inspired glass morphism UI.

### What's Included

✓ Home Feed — Browse curated Pittsburgh news
✓ News Digest — Daily summary of top stories
✓ Saved Articles — Bookmark articles with swipe gestures
✓ Abridged Mode — RSVP speed reader for faster article consumption
✓ Light & Dark Modes — Automatic theme switching with colorblind-friendly accent colors
✓ iOS 26 Glass UI — Modern blur effects, glass buttons, and smooth animations
✓ Gesture Controls — Swipe right to go back, swipe left to save articles
✓ Clean Settings — Customize your reading experience

### What We're Testing

We need your feedback on:
- Stability: Does the app crash or freeze?
- Performance: Is scrolling smooth? Are animations fluid?
- Gestures: Do swipe interactions feel natural?
- Usability: Is navigation intuitive? Are glass buttons easy to identify and tap?
- Design: Is the glass morphism aesthetic appealing? Is typography readable in both themes?
- RSVP Reader: Is the Abridged Mode speed reader effective?

### New in This Build

✨ iOS 26-inspired glass morphism UI components
✨ Swipe gestures for navigation and quick save
✨ Enhanced tab bar with improved blur effects
✨ Dark mode with automatic theme switching
✨ RSVP speed reader (Abridged Mode)
✨ Haptic feedback throughout the app
