# Updates Folder

Last Updated: 2026-01-26

This folder keeps historical release tracking artifacts. The live backlog now lives in the root `todo.md`; use that as the source of truth for in-flight work.

## Contents
- `completed.md` — Archive of finished tasks (by version/phase). Keep adding summaries here when work ships.
- `welcome.md` — Historical dashboard from Jan 16, 2026; helpful context but not the live plan.
- `todo.md` — Historical roadmap (v1.1/v1.2 era). Left for legacy IDs; do not add new tasks here.

## Usage
- Add new tasks to root `todo.md` only; avoid duplicating here.
- When tasks complete, add a short entry to `updates/completed.md` (and reference the TODO ID if applicable).
- If you update these historical files, note the date and reason in a short banner.
